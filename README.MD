# Wine Quality Prediction - MLOps Pipeline

## 业务任务
预测红葡萄酒的质量评分（0-10分），帮助酿酒厂快速评估产品质量。

## 数据说明
- 来源：UCI Machine Learning Repository
- 样本数：1599 条
- 特征数：11 个（固定酸度、挥发性酸度、柠檬酸、残糖、氯化物、游离二氧化硫、总二氧化硫、密度、pH、硫酸盐、酒精）
- 目标变量：quality（质量评分 3-8分）

## 使用的模型
随机森林分类器（RandomForestClassifier）

## 模型性能
- 准确率：70.0%
